Section 1 Virtual Silcton Introduction

Welcome to the documentation for Virtual Silcton.

The Virtual Silcton environment was originally designed in Unity3D by Victor Schinazi and Drew Dara-Abrams. In 2012, Steven Weisberg began updating and maintaining the documentation and website for administration of the Virtual Silcton environment.

This documentation was drafted by Steven M. Weisberg and last edited on the date at the top of this page.
